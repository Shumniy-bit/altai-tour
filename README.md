# Алтай Тур — Экскурсии и отдых в Горном Алтае

Лендинг для туристической компании, организующей авторские туры по Горному Алтаю. Сайт полностью свёрстан на HTML/CSS/JS и готов к переносу на WordPress.

> ⚡ **Статус:** Фронтенд завершён. Проект подготовлен для посадки на WordPress.

## 🔗 Демо

[Посмотреть сайт](https://shumniy-bit.github.io/altai-tour/)

## 🚀 Функционал

- **Адаптивная вёрстка** — 5 точек перелома (360px, 480px, 768px, 992px, 1200px+)
- **Фиксированная шапка** — с изменением размера при скролле
- **Бургер-меню** — плавное открытие/закрытие с overlay
- **Каталог туров** — 6 карточек с баджами (хит продаж, новинка, скидка, сезонное, зимний)
- **Раскрывающийся блок** — «Узнать больше» с анимацией появления
- **Фотоколлаж** — 3 перекрывающихся круга с изображениями
- **Форма обратной связи** — с валидацией и выводом сообщений
- **Кнопка «Наверх»** — с индикатором прогресса скролла
- **Анимация появления** — элементов при прокрутке (Intersection Observer)
- **Плавный скролл** — к секциям с учётом высоты шапки
- **Подсветка активного пункта меню** — при скролле

## 🛠 Технологии

- HTML5 (семантическая вёрстка)
- CSS3 (Flexbox, Grid, CSS Variables, адаптив)
- JavaScript (Vanilla JS, Intersection Observer)
- SVG-спрайты (иконки)


## 🖥 Готовность к посадке на WordPress

Проект полностью подготовлен для интеграции в WordPress:

- **Чистая вёрстка** — HTML, CSS и JS разделены, легко разбиваются на шаблоны темы
- **Секционная структура** — каждая секция может стать отдельным блоком в конструкторе
- **Каталог туров** — карточки готовы к выводу через цикл WordPress (кастомный тип записи)
- **Контактная форма** — легко заменить на Contact Form 7 или WPForms
- **CSS Variables** — единая дизайн-система, цвета и шрифты меняются в одном месте
- **Независимые скрипты** — не конфликтуют с jQuery (если используется в теме)
